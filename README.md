# SwiftUITimeTable

完全基于SwiftUI标准API的时间表，支持iOS/MacOS（WatchOS和TvOS没有测试过效果）

## 特点
- 纯SwiftUI框架
- 支持左滑右滑切换周视图（基于TabView）
- 支持定制各种参数，包括展示一周的天数，时间段等。
